Indedata

April 2016
Indedata is a Ruby on Rails application that allows users to interact with thousands of lines of CSV data to create charts and graphs to analyse Philly's bike share program.

Tap Street

March 2016
Tap Street is a Ruby on Rails application that allows users to interact with Google Maps API and beverage APIs to put down markers that share locations that have unique wines and beers on tap and in bottle!

Straight-forward Bootstrap

January 2016
I created a new website for StevenAveryCase.org using Bootstrap. The goal of the website it to publish documents relating to the trials of Steven Avery, subject of Netflix's Making A Murderer. This is the first website I began to build with the mindset of it being able to fit seamlessly into mobile.

Lancaster City Guide

December 2015
I created a guide to Lancaster, PA using HTML and CSS for a couple getting married there. A few weeks later I returned to the project to add media queries for mobile and tablet sizes. The gallery is made using the float property in CSS. I could improve upon the mobile menu in the future, though I'm happy overall with how the project turned out.
